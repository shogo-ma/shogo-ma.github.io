kotlin入門: android studioでkotlinを使う

May 14, 2016   #android  #android studio  #kotlin 

androidのアプリ開発でkotlinが話題らしいので，触ってみることにした． アプリ開発はあまり経験がないが，この際新しくて面白そうということで色々挑戦してみる．

今回はandroid studioでkotlinのソースコードを実行するまでをやってみる．

流れとしては以下のような感じ．

  • android studioにkotlin pluginを導入
  • 作成したプロジェクトのjavaのソースコードをkotlinに変換する

android studioにkotlin pluginを導入

手順は簡単． - android studioのメニューバーでpreference > Pluginsを選ぶ． - PluginsのInstall JetBrains pluginを選ぶ - kotlinで検索してKotlin pluginをインストール

で終わり．

作成したプロジェクトのjavaのソースコードをkotlinに変換する

こちらはもっと簡単． - android studioのメニューバーでCode > Convert Java File to Kotlin Fileを選択する．

で，元の.javaのファイルが.ktのファイルに変換されます． 後は実行するだけ．場合（自分でもよく分かってない）によっては，オプショナル関連で落ちることがあるけど，それは単純に?をつければいいだけ．
