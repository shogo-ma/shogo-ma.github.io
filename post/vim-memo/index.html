自分用Vimメモ

Jun 5, 2016   #vim 

キーマップ

  • 全てでマップ
    • map(再割り当てあり), noremap(再割り当てなし)
  • insert modeでマップ
    • imap(再割り当てあり), inoremap(再割り当てなし)
  • visual modeでマップ
    • vmap(再割り当てあり), vnoremap(再割り当てなし)
  • command modeでマップ
    • cmap(再割り当てあり), cnoremap(再割り当てなし)
  • normal modeでマップ
    • nmap(再割り当てあり), nnoremap(再割り当てなし)

スコープ

  • ローカルスコープ
    • l:
  • グローバルスコープ
    • g:
  • スクリプトスコープ
    • s:

プラグイン作り

pluginとautoloadの違い

  • pluginは実際にプラグインを使う上での必要最低限のものを記述したvimファイルを配置しておく

    • commandなど

  • autoloadはpluginには置けないような重たい処理を記述したvimファイルを配置する

その他

  • autocmd
    • バッファが作成された時のイベント処理
